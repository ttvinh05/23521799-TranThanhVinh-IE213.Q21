- **Mô tả ngắn gọn cho từng lab:**
    + Lab01: Thiết lập môi trường làm việc với MongoDB (Atlas, Compass) và thực hành các thao tác CRUD cơ bản trên mongosh, bao gồm: thêm dữ liệu, tạo unique index, truy vấn có điều kiện, cập nhật document và sử dụng Aggregation Pipeline để gom nhóm, tính toán dữ liệu.
    + Lab 02: Thiết lập Backend với Node.js và ExpressJS, cấu hình biến môi trường, kết nối cơ sở dữ liệu MongoDB Atlas, xây dựng kiến trúc phân lớp Route - Controller - DAO và truy xuất dữ liệu thông qua API.
    + Lab 03: Hoàn thiện kiến trúc Back-end cho ứng dụng minh họa. Thực hành xây dựng các API thao tác thêm, sửa, xóa review. Thực hiện kiểm thử API bằng phần mềm Insomnia.
    + Lab 04: Thiết lập Frontend cho ứng dụng với ReactJS. Tích hợp UI bằng React-Bootstrap, sử dụng Hook `useState` để quản lý trạng thái đăng nhập trên thanh điều hướng và thiết lập định tuyến chuyển trang bằng `react-router-dom`.
